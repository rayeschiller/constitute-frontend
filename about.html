The worst tweets on Twitter

About

This project displays data visualizations about toxicity on Twitter for various high profile politicians. The data was collected using the Twitter Search API using the politician's first and last name and Twitter handles as search key words. In order to get information about the sentiment of the tweet, we used the Perspective API, an AI powered API used to detect abuse and harassment online. Perspective API has the ability to detect toxicity, identity attack, and sexually explicit attributes, defined below.

Toxicity: Rude, disrespectful, or unreasonable comment that is likely to make people leave a discussion.

Identity Attack: Negative or hateful comments targeting someone because of their identity.

Sexually Explicit: Contains references to sexual acts, body parts, or other lewd content.

Perspective API takes comment text as input and returns a "score" from 0 to 1 that indicates the probability that the comment is similar to toxic comments it's seen in the past. A score of 0 signifies 0% probability that the comment is toxic and a score of 1 signifies 100% probability that the comment is toxic.

It is important to note that there are known unintended biases within the Perspective API.
